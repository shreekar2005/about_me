MoodFlix

As part of a Software Engineering course project, I designed and developed MoodFlix, a web application that recommends movies based on user emotions. The web interface, built using Flask, supports multiple input methods including speech, text, and AI chat ensuring a seamless user experience. The system optimizes recommendation relevance, reducing incorrect suggestions by 28%. This project showcases my expertise in machine learning, natural language processing, and web application development.
